

## NodeJS template

### Getting Started

```
npm i
npm run prod:build
```
navigate to the PORT on your browser and navigate to the API-DOCS Endpoint

```
localhost:5000/api-docs
```

### DOCKER

```
docker build -t octopusx-access .
docker images
docker run -it -p 9000:5000 -d ocxaccess
docker ps

```
